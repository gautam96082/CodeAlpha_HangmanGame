# CodeAlpha Hangman Game

## Description

A console-based Hangman Game developed in Python. The player guesses letters to reveal a hidden programming-related word before running out of attempts.

## Features

* Random word selection
* 250+ Programming and Computer Science words
* ASCII Hangman Art
* Input validation
* Letter tracking
* Win/Lose conditions
* Console-based gameplay

## Technologies Used

* Python 3

## Project Structure

```text
CodeAlpha_HangmanGame/

```

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/gautam96082/CodeAlpha_HangmanGame.git
```

2. Navigate to the project folder:

```bash
cd CodeAlpha_HangmanGame
```

3. Run the program:

```bash
python main.py
```
